In this file, there is the code and raw datasets for the crime by area animated barplot made using data from the FBI's US crime dataset for 2010-2019. 
In addition, the ranks were calculated simply in excel to find which states had the highest crime rates for each year, and for each crime. 
The file named gganimate_code2.R contains the code. 
Source: https://ucr.fbi.gov/crime-in-the-u.s
